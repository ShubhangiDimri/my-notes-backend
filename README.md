A simple Notes application built using **Node.js**, **Express** & **EJS**.  
This mini backend project is made to practice routing, CRUD and templating using EJS.

##  Features
- Add new notes easily using form UI
- View all notes on homepage
- View details of a single note
- Edit/update note anytime
- Delete notes permanently
- **Pin / Unpin Notes** to keep important ones on top
- Notes stored in file system 
- Clean EJS templating based UI pages
- Pure Express.js backend
